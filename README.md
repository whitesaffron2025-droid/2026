# 2026 Campaign Dashboard

Static campaign dashboard connected to Supabase project `tkuivkhghmeljcuqwwdy` and table `2026`.

## Live URL

When GitHub Pages is enabled for the `main` branch root, the site will be available at:

https://whitesaffron2025-droid.github.io/2026/

## Features

- Dashboard KPI cards
- Voter records table
- Search, party filter, outreach filter, and page size control
- Update modal for vote, call, D2D, reach, support, transport, assignment, living place, and remarks
- Automatic reach status logic for will-vote, not-vote, guaranteed, called, and D2D reach
- Party analytics and priority insights
- Copyable report summary
- CSV export
